# Indian House Price Prediction - Machine Learning Project

## -> Project Overview

A comprehensive machine learning project that predicts house prices
across major Indian cities using ensemble algorithms. This project
demonstrates an end-to-end ML pipeline from data generation to web
deployment.

------------------------------------------------------------------------

## -> Problem Statement

### Business Context

The Indian real estate market lacks standardized pricing mechanisms,
leading to price discrepancies and information asymmetry between buyers
and sellers.

### Objectives

-   Create a synthetic dataset representing realistic Indian real estate
    conditions\
-   Build and compare multiple ML algorithms\
-   Achieve prediction accuracy above 90%\
-   Develop an interactive web application for real-time predictions

------------------------------------------------------------------------

## -> Dataset Description

**Total Samples:** 800\
**Cities Covered:** 12 Indian cities\
**Price Range:** ₹40 Lakhs -- ₹20 Crores+

### Features

-   Location\
-   Square Feet (sqft)\
-   Bedrooms\
-   Bathrooms\
-   Parking\
-   Furnishing Status\
-   Property Type\
-   Age of Property\
-   Price in Rupees (Target)

------------------------------------------------------------------------

## -> Technical Approach

### ML Pipeline

Data Generation → Preprocessing → Feature Engineering → Model Training →
Evaluation → Deployment

### Models Used

-   XGBoost 
-   LightGBM
-   Random Forest

------------------------------------------------------------------------

## -> Deployment

-   Backend: Scikit-learn models (.pkl)
-   Frontend: Streamlit Web App
-   Visualization: Plotly

------------------------------------------------------------------------

## -> Conclusion

This project demonstrates how ensemble machine learning models can
accurately predict Indian house prices using synthetic data and strong
feature engineering.
